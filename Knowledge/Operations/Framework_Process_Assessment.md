# Process Assessment Framework

## Objetivo

Detectar ineficiencias operativas.

## Variables

Procesos documentados.

Procesos repetitivos.

Dependencias.

Retrabajos.

Controles.

Automatización.

## Preguntas

¿Existe documentación?

¿Quién aprueba?

¿Qué sucede si una persona falta?

¿Cuánto tiempo demora cada actividad?

## Indicadores

Tiempo de ciclo.

Cantidad de aprobaciones.

Cantidad de errores.

Nivel de automatización.

## Hallazgos

Cuellos de botella.

Duplicación.

Dependencia de personas.

Procesos manuales.

## Salida

Mapa de procesos.

Mapa de mejoras.
