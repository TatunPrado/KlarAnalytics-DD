# Jobs To Be Done Framework

## Objetivo

Comprender qué trabajo o necesidad intenta resolver realmente el cliente al contratar un producto o servicio.

## Definición

Jobs To Be Done sostiene que los clientes no compran productos; contratan soluciones para resolver problemas específicos.

## Alcance

Analiza:

* Necesidades funcionales.
* Necesidades emocionales.
* Necesidades sociales.
* Motivaciones de compra.

No analiza:

* Estructura interna.
* Finanzas.
* Riesgos corporativos.

## Variables de Evaluación

### Trabajo Funcional

¿Qué tarea intenta resolver el cliente?

### Trabajo Emocional

¿Cómo quiere sentirse?

### Trabajo Social

¿Cómo quiere ser percibido?

### Contexto

¿Cuándo surge la necesidad?

### Alternativas

¿Qué utiliza actualmente?

## Preguntas Clave

¿Qué problema intenta resolver el cliente?

¿Qué ocurre antes de comprar?

¿Qué ocurre después de comprar?

¿Qué alternativa utiliza hoy?

¿Qué frustraciones experimenta?

¿Qué resultado considera exitoso?

## Indicadores

Nivel de satisfacción.

Frecuencia de recompra.

Abandono de clientes.

Valor percibido.

## Señales de Alerta

El cliente compra exclusivamente por precio.

Dificultad para explicar el valor ofrecido.

Alta rotación de clientes.

Baja fidelización.

## Hallazgos Posibles

Propuesta de valor mal alineada.

Necesidades no cubiertas.

Segmentos mal definidos.

Oportunidades de innovación.

## Recomendaciones Posibles

Redefinir propuesta de valor.

Crear nuevos productos.

Segmentar mejor clientes.

Modificar mensajes comerciales.
