# Process Assessment Skill

## Objetivo

Aplicar el [Framework_Process_Assessment](../Knowledge/Operations/Framework_Process_Assessment.md) para detectar ineficiencias operativas en el negocio.

## Procedimiento

1. **Identificar procesos.** Listar los procesos críticos del negocio (ventas, producción, despacho, cobranza, compras).

2. **Mapear flujo actual.** Documentar el flujo paso a paso de cada proceso incluyendo entradas, salidas, responsables y tiempos.

3. **Detectar ineficiencias:**

   - **Retrabajos:** actividades que se hacen más de una vez por errores o falta de información.
   - **Duplicaciones:** tareas repetidas por distintas áreas sin valor agregado.
   - **Cuellos de botella:** pasos que limitan la capacidad del proceso.
   - **Tareas manuales:** actividades que podrían ser automatizadas y consumen tiempo excesivo.

4. **Evaluar automatización.** Identificar qué tareas manuales y repetitivas tienen potencial de automatización (software, integraciones, workflows).

5. **Priorizar mejoras.** Jerarquizar oportunidades por impacto en tiempo, costo y calidad.

## Salida

Mapa de procesos actual (flujo), ineficiencias detectadas por tipo, nivel de automatización potencial y oportunidades de mejora priorizadas.
