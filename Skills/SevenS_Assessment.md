# Seven S Assessment Skill

## Objetivo

Aplicar el [Framework_7S](../Knowledge/Operations/Framework_7S.md) para evaluar el alineamiento organizacional y la capacidad de ejecución de la empresa.

## Procedimiento

1. **Evaluar cada elemento 7S:**

   - **Strategy:** existencia, claridad y comunicación de la estrategia.
   - **Structure:** organigrama, definición de roles y responsabilidades.
   - **Systems:** procesos, políticas, controles y sistemas de información.
   - **Shared Values:** valores compartidos, cultura organizacional y propósito.
   - **Skills:** competencias organizacionales y técnicas del equipo.
   - **Staff:** personas clave, talento disponible y capacidad instalada.
   - **Style:** estilo de liderazgo, toma de decisiones y comunicación interna.

2. **Asignar madurez.** Calificar cada elemento en nivel 1–5 según la escala del framework (Caótico → Reactivo → Definido → Gestionado → Optimizado).

3. **Detectar desalineaciones.** Identificar incoherencias entre elementos (ej: estrategia ambiciosa con procesos informales, o cultura reactiva con estructura rígida).

4. **Identificar restricciones organizacionales.** Señalar los elementos con menor madurez que limitan la capacidad de ejecución y crecimiento.

## Salida

Diagnóstico organizacional con mapa 7S, nivel de madurez por elemento, desalineaciones detectadas y restricciones organizacionales priorizadas.
